# MOY Official

品牌官网与产品矩阵入口，对应域名：**moy.com**。

## 站点定位

MOY 是桐鸣科技旗下的企业 AI 增长与经营自动化平台。本网站是 MOY 母品牌的公开对外入口，展示产品矩阵（MOY GEO / MOY API / MOY App）、品牌理念和联系方式。

## 本地启动

```bash
npm install
npm run dev        # http://localhost:5175
```

## 构建

```bash
npm run build      # tsc --noEmit && vite build → dist/
```

## 当前状态

**静态 MVP**。所有内容为占位文本，表单为前端模拟提交流程，无后端对接。
