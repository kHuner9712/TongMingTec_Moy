## 二、数据表：geo_content_topics

新增 Entity：

GeoContentTopic

表名：

geo_content_topics

字段：

- id uuid primary generated
- leadId / lead_id nullable
- brandAssetId / brand_asset_id nullable
- reportId / report_id nullable
- title varchar(300)
- contentType varchar(50)
- targetKeyword varchar(200) nullable
- targetQuestion text nullable
- targetAudience varchar(200) nullable
- searchIntent varchar(50) nullable
- platformSuggestion varchar(200) nullable
- priority varchar(20) default medium
- status varchar(20) default idea
- outline text nullable
- keyPoints jsonb nullable
- referenceMaterials jsonb nullable
- complianceNotes text nullable
- plannedPublishDate date nullable
- actualPublishDate date nullable

- createdAt
- updatedAt

- process_explainer 服务流程
- brand_intro 品牌介绍
- faq FAQ

searchIntent 枚举：

- informational 信息了解
- commercial 商业比较
- navigational 品牌导航
- transactional 购买决策

priority：

- high
- medium
- low

status：

- idea
- planned
- drafting
- reviewing
- approved
- published
- archived

## 五、后端接口：Content Topics

全部为管理接口，需要 JWT，不要 @Public()。

新增：

GET /api/v1/geo-content-topics

查询参数：

- leadId 可选
- brandAssetId 可选
- reportId 可选
- status 可选
- priority 可选
- contentType 可选
- keyword 可选，匹配 title / targetKeyword / targetQuestion
- page 默认 1
- pageSize 默认 20，最大 100

POST /api/v1/geo-content-topics

GET /api/v1/geo-content-topics/:id

PATCH /api/v1/geo-content-topics/:id

PATCH /api/v1/geo-content-topics/:id/status

DELETE /api/v1/geo-content-topics/:id

DELETE 使用软删除：status = archived。
