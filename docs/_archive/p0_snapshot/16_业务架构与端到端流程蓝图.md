# MOY 业务架构与端到端流程蓝图

## 1. 业务架构边界
- 当前正式开发：P0
- 暂不纳入：P1
- 保留规划：P2

## 2. P0 业务模块
AUTH、ORG、USR、CM、LM、OM、CNV、TK、TSK、NTF、CHN、AI、AUD、SYS

## 3. 端到端链路
1. 获客：线索录入/分配
2. 沟通：会话接入/消息处理
3. 谈单：商机推进/结果标记
4. 售后：会话转工单/工单闭环
5. 经营：客户状态维护

## 4. 约束
- 链路实现必须可追溯到 RTM。
- 不得引入 P1/P2 功能作为链路必要节点。
