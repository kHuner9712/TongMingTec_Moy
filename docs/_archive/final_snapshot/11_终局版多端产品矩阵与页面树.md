## 一、文档目的

本文档作为 MOY 终局版的**多端产品矩阵与页面树基线**，用于：

1. 定义终局版多端产品矩阵（管理端、员工端、客户端、移动端）
2. 定义每个端的功能模块与页面清单
3. 定义每个端的权限要求与访问控制
4. 定义完整的页面树与页面路由规则
5. 为前端开发提供统一的页面实现规范
6. 明确与 P0/P1 首期基线的差异与演进路径

**阅读建议：**

- 产品负责人：重点阅读产品矩阵、功能模块、页面清单
- 技术负责人：重点阅读页面树、路由规则、权限要求
- 前端开发：重点阅读页面清单、路由规则、布局规范
- 测试工程师：重点阅读页面功能覆盖、权限矩阵

## 三、术语定义

### 3.1 终端术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 管理端 | Admin Portal | Web 管理后台，供企业管理员、平台管理员使用 |
| 员工端 | Staff Portal | Web 工作台 + 移动端 APP，供企业员工日常办公使用 |
| 客户端 | Customer Portal | 客户自助服务门户，供终端客户使用 |
| 移动端 | Mobile App | iOS/Android 原生 APP，供企业员工移动办公使用 |
| 小程序 | Mini Program | 微信小程序/企业微信小程序，轻量级移动应用 |

### 3.2 页面术语

| 术语 | 定义 |
|------|------|
| 页面树 | 系统所有页面的层级结构，定义页面的父子关系与导航路径 |
| 页面路由 | 页面的 URL 路径定义，支持参数化路由 |
| 页面类型 | 列表页、详情页、表单页、看板页、配置页、监控页等 |
| 页面权限 | 访问页面所需的功能权限与数据权限 |

### 3.3 权限术语

| 术语 | 定义 |
|------|------|
| 页面访问权限 | 用户是否有权访问某个页面 |
| 功能操作权限 | 用户在页面内可执行的操作（创建、编辑、删除等） |
| 数据范围权限 | 用户可访问的数据范围（全部、部门、本人等） |
| 字段级权限 | 用户可查看/编辑的字段范围 |
