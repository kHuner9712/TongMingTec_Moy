## 一、文档目标

本文档定义 MOY 终局版与 P0/P1 版本的差异映射关系，用于：

1. 明确终局版相对于 P0/P1 的增量能力
2. 为版本迭代规划提供依据
3. 支撑增量开发与渐进式交付
4. 确保版本间的兼容性与平滑升级
5. 为测试用例设计提供差异参考

## 三、术语定义

### 3.1 核心术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 差异类型 | Difference Type | 终局版相对于 P0/P1 的变更类型 |
| 增量对象 | Incremental Object | 终局版新增的业务对象 |
| 扩展状态机 | Extended State Machine | 终局版扩展的状态流转规则 |
| 增强接口 | Enhanced API | 终局版新增或增强的接口能力 |
| 新增页面 | New Page | 终局版新增的页面或功能模块 |
| 权限扩展 | Permission Extension | 终局版新增的权限定义 |
| 影响范围 | Impact Scope | 变更影响的系统模块和业务流程 |

### 3.2 差异类型定义

| 差异类型 | 编码 | 说明 |
|----------|------|------|
| 新增 | ADD | 终局版新增的能力，P0/P1 不存在 |
| 扩展 | EXT | 终局版扩展的能力，P0/P1 有基础版本 |
| 增强 | ENH | 终局版增强的能力，P0/P1 功能较弱 |
| 变更 | CHG | 终局版变更的能力，P0/P1 有不同实现 |
| 废弃 | DEP | 终局版废弃的能力，P0/P1 存在但不再使用 |
