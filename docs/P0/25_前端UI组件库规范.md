## 一、文档目的

本文档定义 MOY 系统的前端 UI 组件库规范，作为企业级 AI 原生客户管理系统的前端开发基线，用于：

1. 统一前端组件库选型和技术栈
2. 定义组件命名和使用规范
3. 规范主题、样式、交互一致性
4. 提高前端开发效率和代码质量
5. 确保用户体验一致性

## 二、设计原则

### 2.1 核心设计原则

| 原则 | 说明 | 实践要求 |
|------|------|----------|
| **一致性** | 相同功能使用相同交互方式 | 统一使用规范组件，禁止自定义相似功能组件 |
| **可预测** | 用户操作结果可预期 | 遵循用户心智模型，操作前有提示，操作后有反馈 |
| **高效性** | 减少用户操作步骤 | 提供快捷操作、批量操作、智能默认值 |
| **容错性** | 防止用户犯错 | 危险操作二次确认，提供撤销功能 |
| **可访问性** | 支持所有用户访问 | 符合WCAG 2.1 AA标准 |

### 2.2 视觉设计原则

| 原则 | 说明 |
|------|------|
| **层级分明** | 通过颜色、大小、间距区分信息层级 |
| **留白适度** | 保持页面呼吸感，避免信息过载 |
| **对齐统一** | 所有元素遵循统一的网格对齐 |
| **色彩克制** | 主色不超过3种，强调色用于关键操作 |

### 2.3 组件设计原则

| 原则 | 说明 |
|------|------|
| **单一职责** | 每个组件只做一件事 |
| **可组合** | 小组件组合成大组件 |
| **可配置** | 通过Props控制行为和样式 |
| **可扩展** | 预留扩展点，支持自定义 |

### 2.4 禁止重复造轮子原则

| 场景 | 规范 |
|------|------|
| 基础组件 | 必须使用 Ant Design 组件，禁止自定义 Button、Input、Select 等 |
| 图标 | 优先使用 @ant-design/icons，缺失时再自定义 |
| 工具函数 | 优先使用 lodash-es、dayjs，禁止重复实现 |
| Hooks | 优先使用 ahooks，常用场景已有实现 |
| 样式 | 优先使用 Tailwind CSS 原子类，避免自定义 CSS |

### 2.5 组件复用原则

| 原则 | 说明 |
|------|------|
| 三次原则 | 相同代码出现3次以上，必须抽取为公共组件 |
| 业务无关优先 | 优先抽取业务无关的通用组件 |
| 接口稳定 | 公共组件接口变更需评审，保持向后兼容 |
| 文档完善 | 公共组件必须有使用文档和示例 |

```tsx
// components/business/StandardTable/index.tsx
import { Table, TableProps } from 'antd';
import type { ColumnsType } from 'antd/es/table';

interface StandardTableProps<T> extends Omit<TableProps<T>, 'pagination'> {
  columns: ColumnsType<T>;
  dataSource: T[];
  loading?: boolean;
  pagination?: {
    current: number;
    pageSize: number;
    total: number;
    onChange: (page: number, pageSize: number) => void;
  };
  rowKey: string | ((record: T) => string);
}

export function StandardTable<T extends Record<string, unknown>>({
  columns,
  dataSource,
  loading,
  pagination,
  rowKey,
  ...restProps
}: StandardTableProps<T>) {
  return (
    <Table<T>
      columns={columns}
      dataSource={dataSource}
      loading={loading}
      rowKey={rowKey}
      pagination={
        pagination
          ? {
              current: pagination.current,
              pageSize: pagination.pageSize,
              total: pagination.total,
              onChange: pagination.onChange,
              showSizeChanger: true,
              showQuickJumper: true,
              showTotal: (total) => `共 ${total} 条`,
              pageSizeOptions: ['10', '20', '50', '100'],
            }
          : false
      }
      scroll={{ x: 'max-content' }}
      {...restProps}
    />
  );
}
```

#### 4.2.1 标准表单配置

```tsx
// components/business/StandardForm/index.tsx
import { Form, FormProps, Button, Space } from 'antd';
import { ReactNode } from 'react';

interface StandardFormProps extends FormProps {
  children: ReactNode;
  onSubmit: (values: unknown) => Promise<void>;
  submitText?: string;
  cancelText?: string;
  onCancel?: () => void;
  loading?: boolean;
  footer?: ReactNode;
}

export function StandardForm({
  children,
  onSubmit,
  submitText = '提交',
  cancelText = '取消',
  onCancel,
  loading,
  footer,
  ...restProps
}: StandardFormProps) {
  const [form] = Form.useForm();

  const handleSubmit = async (values: unknown) => {
    await onSubmit(values);
  };

  return (
    <Form form={form} layout="vertical" onFinish={handleSubmit} {...restProps}>
      {children}
      {footer !== null && (
        <Form.Item>
          <Space>
            <Button type="primary" htmlType="submit" loading={loading}>
              {submitText}
            </Button>
            {onCancel && (
              <Button onClick={onCancel}>{cancelText}</Button>
            )}
          </Space>
        </Form.Item>
      )}
    </Form>
  );
}
```

### 4.4 搜索筛选组件

```tsx
// components/business/SearchFilter/index.tsx
import { Form, Input, Select, Button, Space, Row, Col } from 'antd';
import { SearchOutlined, ReloadOutlined } from '@ant-design/icons';

interface FilterField {
  name: string;
  label: string;
  type: 'input' | 'select' | 'date' | 'dateRange';
  options?: { label: string; value: string | number }[];
  placeholder?: string;
}

interface SearchFilterProps {
  fields: FilterField[];
  onSearch: (values: Record<string, unknown>) => void;
  onReset: () => void;
  loading?: boolean;
}

export function SearchFilter({ fields, onSearch, onReset, loading }: SearchFilterProps) {
  const [form] = Form.useForm();

  const handleSearch = () => {
    const values = form.getFieldsValue();
    onSearch(values);
  };

  const handleReset = () => {
    form.resetFields();
    onReset();
  };

  return (
    <Form form={form} layout="inline">
      <Row gutter={16}>
        {fields.map((field) => (
          <Col key={field.name} span={6}>
            <Form.Item name={field.name} label={field.label}>
              {field.type === 'input' && (
                <Input placeholder={field.placeholder || `请输入${field.label}`} allowClear />
              )}
              {field.type === 'select' && (
                <Select
                  placeholder={field.placeholder || `请选择${field.label}`}
                  options={field.options}
                  allowClear
                />
              )}
              {field.type === 'date' && (
                <DatePicker style={{ width: '100%' }} placeholder={field.placeholder} />
              )}
              {field.type === 'dateRange' && (
                <RangePicker style={{ width: '100%' }} />
              )}
            </Form.Item>
          </Col>
        ))}
        <Col span={6}>
          <Space>
            <Button type="primary" icon={<SearchOutlined />} onClick={handleSearch} loading={loading}>
              搜索
            </Button>
            <Button icon={<ReloadOutlined />} onClick={handleReset}>
              重置
            </Button>
          </Space>
        </Col>
      </Row>
    </Form>
  );
}
```

```tsx
const STAGE_CONFIG = [
  { key: 'initial', title: '初步接触', probability: 10 },
  { key: 'requirement', title: '需求确认', probability: 30 },
  { key: 'proposal', title: '方案报价', probability: 50 },
  { key: 'negotiation', title: '商务谈判', probability: 70 },
  { key: 'won', title: '成交', probability: 100 },
];

interface OpportunityStageBarProps {
  currentStage: string;
  status?: 'process' | 'wait' | 'finish' | 'error';
  onStageClick?: (stage: string) => void;
}

export function OpportunityStageBar({ currentStage, status, onStageClick }: OpportunityStageBarProps) {
  const currentIndex = STAGE_CONFIG.findIndex(s => s.key === currentStage);
  
  const getStatus = (index: number): StepsProps['status'] => {
    if (status === 'error' && index === currentIndex) return 'error';
    if (index < currentIndex) return 'finish';
    if (index === currentIndex) return 'process';
    return 'wait';
  };

  return (
    <Steps
      current={currentIndex}
      status={status}
      items={STAGE_CONFIG.map((stage, index) => ({
        key: stage.key,
        title: stage.title,
        description: `${stage.probability}%`,
        status: getStatus(index),
        onClick: () => onStageClick?.(stage.key),
      }))}
    />
  );
}
```

```tsx
import { useAuthStore } from '@/stores/useAuthStore';

interface PermissionGuardProps {
  permission: string | string[];
  mode?: 'any' | 'all';
  fallback?: React.ReactNode;
  children: React.ReactNode;
}

export function PermissionGuard({
  permission,
  mode = 'any',
  fallback = null,
  children,
}: PermissionGuardProps) {
  const { hasPermission } = useAuthStore();
  
  const permissions = Array.isArray(permission) ? permission : [permission];
  
  const hasAccess = mode === 'all'
    ? permissions.every(p => hasPermission(p))
    : permissions.some(p => hasPermission(p));
  
  return hasAccess ? <>{children}</> : <>{fallback}</>;
}
```
