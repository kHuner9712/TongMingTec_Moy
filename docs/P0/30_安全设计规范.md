## 一、文档目的

本文档定义 MOY 系统的安全设计规范，确保：

1. 多租户数据隔离安全
2. 用户身份认证与授权安全
3. 数据传输与存储安全
4. API接口安全防护
5. 敏感数据保护
6. 安全审计与合规

## 三、身份认证安全

### 3.1 JWT Token 设计

#### 3.1.1 Token 结构

```json
{
  "header": {
    "alg": "RS256",
    "typ": "JWT",
    "kid": "key-001"
  },
  "payload": {
    "sub": "user-123",
    "org_id": "org-001",
    "roles": ["sales_rep"],
    "permissions": ["customer:read", "customer:write"],
    "iat": 1712304000,
    "exp": 1712307600,
    "jti": "token-uuid-001"
  },
  "signature": "..."
}
```

#### 3.1.2 Token 配置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 算法 | RS256 | 非对称加密 |
| Access Token 有效期 | 1小时 | 短期有效 |
| Refresh Token 有效期 | 7天 | 长期有效 |
| 签名密钥 | RSA 2048位 | 私钥签名，公钥验证 |
| 存储位置 | HttpOnly Cookie | 防止XSS攻击 |

#### 3.1.3 Token 刷新机制

```typescript
async function refreshToken(refreshToken: string) {
  const payload = verifyRefreshToken(refreshToken);
  
  if (payload.exp < Date.now() / 1000) {
    throw new TokenExpiredError();
  }
  
  const user = await getUserById(payload.sub);
  
  if (user.status !== 'active') {
    throw new UserInactiveError();
  }
  
  const newAccessToken = generateAccessToken(user);
  const newRefreshToken = generateRefreshToken(user);
  
  await revokeRefreshToken(refreshToken);
  
  return { accessToken: newAccessToken, refreshToken: newRefreshToken };
}
```

### 3.2 密码安全

#### 3.2.1 密码复杂度要求

| 要求 | 说明 |
|------|------|
| 最小长度 | 8位 |
| 最大长度 | 64位 |
| 复杂度 | 必须包含大小写字母、数字、特殊字符中的3种 |
| 历史检查 | 不能与最近5次密码相同 |
| 弱密码检查 | 不能使用常见弱密码 |

#### 3.2.2 密码存储

```typescript
import * as bcrypt from 'bcrypt';

const SALT_ROUNDS = 12;

async function hashPassword(password: string): Promise<string> {
  return bcrypt.hash(password, SALT_ROUNDS);
}

async function verifyPassword(password: string, hash: string): Promise<boolean> {
  return bcrypt.compare(password, hash);
}
```

#### 3.2.3 密码重置流程

```
1. 用户请求重置密码
2. 系统生成重置Token（有效期15分钟）
3. 发送重置链接到用户邮箱
4. 用户点击链接，验证Token
5. 用户设置新密码
6. 系统更新密码，撤销所有Token
7. 发送密码变更通知
```

```typescript
async function applyDataScope(query: SelectQueryBuilder, user: User, resource: string) {
  const dataScope = await getUserDataScope(user.id, resource);
  
  switch (dataScope.type) {
    case 'all':
      break;
    case 'department':
      query.andWhere('entity.department_id = :deptId', { deptId: user.departmentId });
      break;
    case 'department_and_sub':
      const deptIds = await getSubDepartmentIds(user.departmentId);
      query.andWhere('entity.department_id IN (:...deptIds)', { deptIds });
      break;
    case 'self':
      query.andWhere('entity.owner_id = :userId', { userId: user.id });
      break;
    case 'custom':
      query.andWhere('entity.id IN (:...ids)', { ids: dataScope.entityIds });
      break;
  }
  
  return query;
}
```

## 六、API 安全防护

### 6.1 请求限流

#### 6.1.1 限流策略

| 接口类型 | 限流规则 | 说明 |
|----------|----------|------|
| 登录接口 | 5次/分钟/IP | 防暴力破解 |
| 短信接口 | 1次/分钟/手机号 | 防短信轰炸 |
| API接口 | 100次/分钟/用户 | 正常业务 |
| 批量接口 | 10次/分钟/用户 | 批量操作 |

#### 6.1.2 限流实现

```typescript
@Injectable()
export class RateLimitGuard implements CanActivate {
  constructor(private readonly redis: RedisService) {}
  
  async canActivate(context: ExecutionContext): Promise<boolean> {
    const request = context.switchToHttp().getRequest();
    const key = this.getRateLimitKey(request);
    const limit = this.getRateLimit(request);
    
    const current = await this.redis.incr(key);
    
    if (current === 1) {
      await this.redis.expire(key, 60);
    }
    
    if (current > limit) {
      throw new TooManyRequestsException('请求频率超限，请稍后再试');
    }
    
    request.res.setHeader('X-RateLimit-Limit', limit);
    request.res.setHeader('X-RateLimit-Remaining', Math.max(0, limit - current));
    
    return true;
  }
}
```

```typescript
import * as xss from 'xss';

function sanitizeInput(input: string): string {
  return xss(input, {
    whiteList: {
      a: ['href', 'title'],
      b: [],
      i: [],
      u: [],
    },
  });
}

@Pipe()
export class SanitizePipe implements PipeTransform {
  transform(value: any) {
    if (typeof value === 'string') {
      return sanitizeInput(value);
    }
    if (typeof value === 'object' && value !== null) {
      Object.keys(value).forEach(key => {
        if (typeof value[key] === 'string') {
          value[key] = sanitizeInput(value[key]);
        }
      });
    }
    return value;
  }
}
```

### 7.2 数据脱敏

#### 7.2.1 脱敏规则

| 数据类型 | 脱敏规则 | 示例 |
|----------|----------|------|
| 手机号 | 中间4位 | 138****1234 |
| 邮箱 | @前部分 | a***@example.com |
| 身份证 | 中间10位 | 110***********1234 |
| 银行卡 | 中间12位 | 6222************1234 |

#### 7.2.2 脱敏实现

```typescript
function maskPhone(phone: string): string {
  if (!phone || phone.length < 7) return phone;
  return phone.replace(/(\d{3})\d{4}(\d{4})/, '$1****$2');
}

function maskEmail(email: string): string {
  if (!email || !email.includes('@')) return email;
  const [local, domain] = email.split('@');
  const maskedLocal = local.charAt(0) + '***';
  return `${maskedLocal}@${domain}`;
}

function maskIdCard(idCard: string): string {
  if (!idCard || idCard.length < 15) return idCard;
  return idCard.replace(/(\d{3})\d{10}(\d{4})/, '$1***********$2');
}
```

#### 7.3.1 加密算法选择

| 场景 | 算法 | 说明 |
|------|------|------|
| 密码存储 | bcrypt | 单向哈希 |
| 敏感数据加密 | AES-256-GCM | 对称加密 |
| Token签名 | RS256 | 非对称加密 |
| 传输加密 | TLS 1.3 | 传输层加密 |

#### 7.3.2 敏感数据加密实现

```typescript
import * as crypto from 'crypto';

const ALGORITHM = 'aes-256-gcm';
const KEY = Buffer.from(process.env.ENCRYPTION_KEY, 'hex');

function encrypt(plaintext: string): string {
  const iv = crypto.randomBytes(12);
  const cipher = crypto.createCipheriv(ALGORITHM, KEY, iv);
  
  let encrypted = cipher.update(plaintext, 'utf8', 'hex');
  encrypted += cipher.final('hex');
  
  const authTag = cipher.getAuthTag();
  
  return `${iv.toString('hex')}:${authTag.toString('hex')}:${encrypted}`;
}

function decrypt(ciphertext: string): string {
  const [ivHex, authTagHex, encrypted] = ciphertext.split(':');
  
  const iv = Buffer.from(ivHex, 'hex');
  const authTag = Buffer.from(authTagHex, 'hex');
  
  const decipher = crypto.createDecipheriv(ALGORITHM, KEY, iv);
  decipher.setAuthTag(authTag);
  
  let decrypted = decipher.update(encrypted, 'hex', 'utf8');
  decrypted += decipher.final('utf8');
  
  return decrypted;
}
```
