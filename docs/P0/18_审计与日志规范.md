# MOY 审计与日志规范

## 1. 审计目标
- 所有关键业务写操作可追溯。
- 支持故障排查与合规审计。

## 2. 审计字段
- `request_id`
- `org_id`
- `operator_id`
- `action`
- `target_type`
- `target_id`
- `result`
- `created_at`

## 3. 审计触发点
- 客户状态变更
- 线索转化
- 商机结果标记
- 会话转接/关闭
- 工单流转
- 权限配置变更

## 4. 存储
- 统一落库：`TABLE-AUD-001 audit_logs`
