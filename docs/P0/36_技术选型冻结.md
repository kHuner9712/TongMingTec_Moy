## 2. P0 冻结技术栈
| 层级 | 技术 |
| --- | --- |
| 前端 | React + TypeScript + Ant Design |
| 后端 | NestJS + TypeScript |
| 数据库 | PostgreSQL 15+ |
| 缓存 | Redis 7+ |
| 实时通信 | WebSocket |
| 网关 | API Gateway |
| 日志 | 结构化日志 + 审计日志 |

## 3. 关键技术决策
- 首页路由统一为 `/dashboard`。
- API 路径参数统一 `{id}`。
- 商机结果枚举统一 `won/lost`。
- AI 仅启用智能回复与任务查询。

## 4. 当前不纳入（P1/P2）
- 知识库向量检索
- 数据看板引擎
- 自动化规则引擎
- 移动端技术栈

## 5. 冻结约束
- 未经评审不得替换核心框架。
- 新技术引入必须先更新 HLD/DBD/API/RTM。
