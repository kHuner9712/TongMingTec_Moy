## 4. 统一硬约束
- 编号：REQ/PAGE/API/PERM/TEST/ACPT/TASK/TABLE/SM。
- 路径参数一律 `{id}`。
- `/` 仅重定向到 `/dashboard`。
- 商机结果仅 `won/lost`。
- 会话权限必须 `SEND/ACCEPT/TRANSFER/CLOSE` 细分。
- 多租户隔离以 `org_id` 为核心。
