# MOY 租户初始化配置清单

## 1. 初始化范围（P0）
- 组织基础信息
- 默认角色与权限
- 默认渠道配置
- 默认系统参数

## 2. 初始化项
| 项目 | 说明 |
| --- | --- |
| organization | 创建组织记录（TABLE-ORG-001） |
| roles | 初始化 ROLE-ADMIN/SALES/SERVICE |
| permissions | 初始化 PERM 清单 |
| channels | 初始化默认渠道 |
| system_configs | 初始化系统阈值参数 |

## 3. 验收
- 初始化后可直接完成登录与基础业务操作。
