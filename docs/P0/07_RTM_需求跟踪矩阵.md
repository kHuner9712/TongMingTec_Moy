## 2. RTM 规则
- 仅跟踪 `P0` 需求。
- 每条 `REQ` 必须闭环映射：`PAGE/API/TABLE/SM/PERM/TEST/ACPT/TASK`。
- 存在缺口时不得宣告“可全量开发”。
