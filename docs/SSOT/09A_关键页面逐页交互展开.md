# MOY 关键页面逐页交互展开

## 1. 文档定位
本文档是 [09_页面与交互规格.md](C:/Users/15864/Desktop/TongMingTech_Moy/docs/SSOT/09_页面与交互规格.md) 的实现级子文档。`09` 保留 PAGE 主索引与全局规则；本文负责把终局关键页面补到可直接生成 React 页面与交互测试的粒度。

约束：
- 同一 PAGE 事实仍归属 `09` 主链；本文是其实现级展开，不形成第二套 PAGE 来源。
- 所有页面都默认具备 `loading / empty / error / forbidden / degraded` 五态。
- `degraded` 统一表示 WS 失败后回退轮询，不允许整页无反馈。

记法：
- `基本信息` 中统一包含 `path / terminal / introduced_in / required_in / release_scope / status / goal / api / perm / sm / ws`。
- `合同` 中统一包含 `区块 / 列表或表单 / 动作规则 / 刷新策略`。
