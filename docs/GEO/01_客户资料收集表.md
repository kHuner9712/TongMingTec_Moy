# MOY GEO 客户资料收集表

## 1. 文档用途

本文档用于在 GEO 服务启动前系统化收集客户信息。填写越完整，诊断越准确，内容生产越高效。

- 填写方式：由 GEO 团队引导客户逐项填写，或客户自行填写后由 GEO 团队补充确认。
- 机密声明：本表内容仅用于 GEO 诊断与内容生产，未经客户书面授权不向第三方披露。
