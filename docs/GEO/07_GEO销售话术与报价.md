## 2. 3 分钟介绍话术

### 场景：初次接触，客户问"你们是做什么的？"

```
MOY GEO 是 AI 搜索增长与品牌可见度服务——

简单说，就是帮助企业在 AI 时代被看见。

现在越来越多的用户直接用 ChatGPT、豆包、Kimi 这些 AI 搜索和问答，
不再打开百度或 Google 搜网页了。
当用户在 AI 里问"哪个 CRM 好用"或者"上海有什么好的装修公司"的时候，
如果你的品牌从来没在回答里出现过 — 你已经在丢客户了。

我们做的不是传统 SEO，而是 GEO——让 AI 模型能读懂你的品牌，
知道你是谁、做什么、为什么值得推荐。
我们基于你的真实信息，帮你建设面向 AI 的结构化内容资产，
然后每月监测 AI 是怎么描述你的，持续优化。

目前有三个服务级别：第一级是做一个全面诊断，告诉你现在的可见度是多少分；
如果要持续做，标准版是 9,800 一个月，我们帮你建内容、做监测、每月复盘。
```

### 场景：客户问"GEO 和 SEO 有什么区别？"

```
SEO 是优化网页让搜索引擎排名靠前——目标是链接排在搜索结果第一页。
GEO 是优化品牌信息让 AI 模型理解和推荐——目标是品牌出现在 AI 的回答里。

两者的方法论完全不同：
- SEO 做关键词密度、外链、域名权重
- GEO 做结构化品牌信息、AI 可读的 FAQ 和知识库、Schema 标记

而且 SEO 的成果可能因为 AI 搜索的崛起而贬值——
用户不再点链接了，直接在 AI 里获得答案。
但 GEO 恰恰是应对这个变化的。
```

## 3. 客户常见问题（FAQ）

### Q1：你们能保证我的品牌被 AI 推荐吗？

```
我们不能保证，也不应该保证。

AI 模型的回答受多种因素影响：模型更新、训练数据变化、检索算法调整等等。
任何一个跟你保证"一定让你排第一"的公司，你都要警惕。

我们能做的是：基于你的真实品牌信息，
在 AI 最可能检索的地方建设高质量、结构化的内容，
显著提高你被看见的概率，并通过月度监测来追踪实际效果。
```

### Q2：做的内容发在哪里？

```
首选是你自己的官网——这是最权威的信息源。
我们会优化你官网的 FAQ 页面、帮助中心、Blog、品牌介绍页。
同时我们会给分发建议，比如哪些第三方平台适合你的行业发布内容。

我们不帮你批量注册垃圾账号去发内容。
我们做的每一篇内容都应该是你品牌资产的一部分。
```

### Q3：多久能看到效果？

```
诊断报告是 3-5 个工作日就出来——这是立刻能看到的。
内容发布后，AI 模型收录新信息通常需要几周到几个月不等。
我们一般建议客户至少签约 3 个月以上才能看到趋势变化。
第一份月度报告是签约后第三周开始做，第四周交付。
```

### Q4：你们做过哪些行业的案例？

```
我们目前主要服务几类客户：
- 企业服务/SaaS 公司
- 本地服务商（装修、家政、医美等）
- 希望提升 AI 搜索曝光的品牌方

具体的客户案例需要和客户确认后才能分享。
你可以先看看我们免费做的诊断报告模板，了解我们分析问题的深度。
```

### Q5：和 xxx（竞品）有什么区别？

```
我们不做大量低质量内容批量发布。
我们不做黑帽技术去欺骗 AI。
我们坚持基于客户真实信息做品牌建设。

另外，MOY GEO 是 MOY 品牌体系的一部分——
MOY 下面还有 MOY API（给开发者用的多模型网关）和 MOY App（客户经营系统）。
我们的 GEO 团队对 AI 模型的技术理解是我们的核心差异。
```

### Q6：标准版 9,800/月，具体包含什么？

```
标准版的月度交付包含：
1. 首次品牌事实资产建设（一次性）
2. 每月 ≥4 篇 GEO 内容（FAQ、行业问答、案例等）
3. 月度全平台 AI 可见度监测
4. 月度复盘报告
5. 专属 GEO 顾问对接

以一个月 4 篇为例，相当于每篇内容背后有诊断、选题、写作、Schema 部署、效果追踪的全流程。
如果客户内容需求更多，可以升级到定制版。
```

## 5. 成交建议

### 5.1 跟进节奏

| 时间点 | 动作 |
| --- | --- |
| 初次接触当天 | 发送 3 分钟介绍话术 + GEO 不是什么（合规声明） |
| 次日 | 发送 1-2 个诊断报告示例（隐去客户信息），展示专业度 |
| 客户表达意向后 | 发送客户资料收集表，启动免费初步咨询 |
| 诊断报告完成 | 安排 30 分钟诊断解读会（线上） |
| 解读会后 | 24 小时内发送定制化方案和报价 |
| 签约后 | 3 个工作日内启动品牌事实资产建设 |

### 5.2 成交信号

- 客户主动问"这个怎么收费"→ 立即引导到诊断报告环节
- 客户问"做了多久能看到效果"→ 说明客户已在做决策，给案例和数据
- 客户问"竞品也在做吗"→ 说明紧迫感，展示竞品可见度对比

## 6. 严禁承诺事项

以下事项在销售过程中**绝对不能承诺**：

| 禁止承诺 | 原因 |
| --- | --- |
| "保证你的品牌在 AI 中被推荐" | AI 行为不可控 |
| "百分百提高可见度" | 没有百分百 |
| "1 个月就看到明显效果" | 不切实际 |
| "我们认识 XX 平台的内部人员" | 虚假 |
| "AI 的数据我们都能控制" | 不属实 |
| "比竞品一定更好" | 不可验证的承诺 |
| "不签就会错过" | 制造焦虑 |
| "我们有秘密技术" | 不透明 |
