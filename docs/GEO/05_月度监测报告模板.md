# MOY GEO 月度监测报告

## 1. 报告用途

月度监测报告是标准版和定制版客户的月度交付物，用于：

- 追踪品牌在 AI 平台中可见度的月度变化
- 总结本月新增的内容资产
- 对比竞品动态
- 发现并修正 AI 回答中的错误信息
- 规划下月优化方向

输出格式：PDF（对外交付）+ Markdown（内部存档）。
