# MOY GEO 内容选题库模板

## 1. 选题库用途

GEO 内容选题库用于规划和追踪品牌 AI 可读内容的生产。选题来源：

- AI 可见度诊断中发现的**信息缺口**
- 客户提供的**常问问题**
- 目标客户可能在 AI 中提出的**行业问题**
- 竞品已有的、但客户品牌缺失的**内容类型**

## 2. 选题分类体系

GEO 内容按以下 10 类规划，每类有不同目标和格式：

| 分类 | 目标 | 典型格式 | AI 平台消费场景 |
| --- | --- | --- | --- |
| 行业问答 | 让 AI 在行业认知问题中引用品牌 | FAQ / 短知识 | "XX 行业有哪些趋势" |
| 本地服务 | 让 AI 在地域性推荐中出现品牌 | 结构化介绍 | "北京有哪些好的 XX 服务商" |
| 竞品对比 | 让 AI 在对比类问题中客观呈现差异 | 对比表 / 说明 | "A 和 B 有什么区别" |
| 购买决策 | 帮助潜在客户选型时获取信息 | 选购指南 | "怎么选 XX 服务商" |
| 常见误区 | 纠正行业误解，建立专业形象 | 辟谣 / 科普 | "关于 XX 的常见误区" |
| 案例拆解 | 用真实案例展示解决问题的能力 | 案例故事 | "XX 行业是怎么解决 YY 问题的" |
| 价格解释 | 透明化价格构成，降低决策门槛 | 价格说明 | "XX 服务一般多少钱" |
| 服务流程 | 让客户理解合作方式 | 流程图 / 步骤 | "和 XX 合作是怎样的流程" |
| 品牌介绍 | 让 AI 准确描述品牌 | 结构化品牌页 | "[品牌名]是做什么的" |
| FAQ | 直接回答客户高频问题 | 问答对 | 各种直接提问 |

## 3. 选题登记表

每条选题登记以下信息：

| 字段 | 说明 |
| --- | --- |
| 选题编号 | GEO-TOPIC-[年月]-[序号] |
| 选题分类 | 从上述 10 类中选择 |
| 选题标题 | 面向 AI 和用户的内容标题 |
| 目标关键词 | 希望 AI 关联的核心词 3-5 个 |
| 目标平台 | 主要面向哪些 AI 平台 |
| 内容摘要 | 50 字概括内容要点 |
| 素材来源 | 参考的事实资产包 / 客户资料 |
| 优先级 | P0(紧急) / P1(高) / P2(中) / P3(低) |
| 状态 | 待生产 / 生产中 / 待审核 / 已发布 |
| 负责人 | |
| 计划日期 | |
| 完成日期 | |
| 发布链接 | 内容发布后的 URL |

## 5. 生产节奏建议

| 套餐 | 月度产能 | 节奏 |
| --- | --- | --- |
| 标准版 | ≥4 篇/月 | 每周至少 1 篇 |
| 定制版 | ≥10 篇/月 | 每周 2-3 篇 |

## 6. 内容质量标准

每篇 GEO 内容需满足：

- [ ] 基于品牌事实资产包中的真实信息
- [ ] 结构清晰：有标题、小标题、要点
- [ ] Schema 标记完整（FAQPage / Article / HowTo）
- [ ] 无夸大、无虚假、无违反合规声明的表述
- [ ] 经客户审核确认后方可发布
- [ ] 发布后记录 URL，进入月度监测追踪
