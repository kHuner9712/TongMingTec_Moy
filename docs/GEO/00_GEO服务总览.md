# MOY GEO 服务总览

## 1. 文档定位

本文档是 MOY GEO 服务化交付体系的唯一总入口，统一定义：

- MOY GEO 的产品定位与核心价值
- 服务范围与交付物清单
- 合规底线与不做事项
- 文档导航（指向各模板与 SOP）

适用对象：GEO 交付团队、销售团队、客户。

## 2. MOY GEO 是什么

**MOY GEO 是 MOY 旗下的 AI 搜索增长与品牌可见度服务。**

帮助企业在 ChatGPT、豆包、Kimi、通义千问、Perplexity、Google AI Overviews、Bing Copilot 等 AI 搜索/问答环境中被看见、被理解、被推荐。

### 核心价值

| 价值点 | 说明 |
| --- | --- |
| 被看见 | 品牌信息出现在 AI 的回答中，而非缺席 |
| 被理解 | AI 对品牌的描述准确、完整、正面 |
| 被推荐 | 在用户询问"有哪些好的 XX 服务商"时被优先引用 |

### 不是什么

| 不是 | 说明 |
| --- | --- |
| 不是 SEO 外包 | GEO 不优化传统搜索引擎排名，而是优化 AI 模型对品牌的认知 |
| 不是软文/水军 | 不做垃圾内容批量发布，不做虚假评论 |
| 不是黑帽操纵 | 不通过漏洞或欺骗手段操控 AI 回答 |
| 不是一次性服务 | 品牌在 AI 中的可见度需要持续维护和优化 |

## 4. 合规底线

MOY GEO 的所有服务必须遵守以下原则：

### 4.1 必须做的事

- 基于客户真实资料、真实服务和真实案例进行内容建设
- 所有对外发布的内容需经客户审核确认
- 引用数据需标注来源和时间
- 客户案例需取得客户书面授权

### 4.2 不允许做的事

| 禁止事项 | 说明 |
| --- | --- |
| 虚假宣传 | 不夸大产品功能、不编造服务能力 |
| 伪造案例 | 不编造客户成功案例、不伪造效果数据 |
| 伪造媒体引用 | 不伪造媒体报道、不编造第三方评价 |
| 垃圾内容批量发布 | 不做大量低质量、重复内容的批量发布 |
| 恶意操纵搜索结果 | 不通过技术手段欺骗 AI 模型 |
| 侵犯竞品权益 | 不对竞品进行不实描述或恶意诋毁 |
| 承诺排名 | 不承诺"一定让 AI 推荐你""保证排第一" |
| 收费后不交付 | 每个阶段有明确交付物，先诊断后收费 |
