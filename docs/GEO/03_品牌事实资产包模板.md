# MOY GEO 品牌事实资产包

## 1. 资产包用途

品牌事实资产包是 GEO 内容生产的核心素材库。它将客户品牌的关键信息按 AI 可读的结构化格式整理，确保：

- 所有内容生产基于统一、准确的事实源
- AI 爬取时能有效提取和理解品牌信息
- 客户内部对齐对外传播的标准口径

## 2. 使用规则

1. 资产包由 GEO 团队基于客户资料表整理，由客户审核确认后方可使用。
2. 每次内容生产前，对照资产包确保信息准确。
3. 资产包至少每季度更新一次，重大变化随时更新。
4. 标记为"不可使用"的材料，在任何情况下不得对外引用。
